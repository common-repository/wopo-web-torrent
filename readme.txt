== Description ==

WoPo Web Torrent is a streaming torrent client for the web browser 

You can add shortcode [wopo-web-torrent] to page you want to show WoPo Web Torrent form. To get started, use the file form button and select files to share from your computer. Copy the Share Link and send it to your friends. Keep open your browser when your friends download it.

You can use Magnet URI or Torrent Hash for download form.

Demo Online: [wopofile.xyz](https://wopofile.xyz/ "WoPo File")

Icons made by [Smashicons](https://www.flaticon.com/authors/smashicons "Smashicons") from [www.flaticon.com](https://www.flaticon.com/ "Flaticon")

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wopo-web-torrent` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Add shortcode [wopo-web-torrent] to page you want to show app
